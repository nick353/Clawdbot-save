# Skill Triggers - 日常会話からの自動トリガー

このファイルは、リッキーが日常会話の中で自然にSkillsを使用するためのガイドっぴ。

## 🎯 基本方針
1. **ユーザーは「〜のSkillを使って」とは言わない**
2. **会話の文脈から自動的に適切なSkillを判断**
3. **タスクに役立ちそうなSkillがあれば自由に追加OK**
4. **追加したSkillも即座にトリガー設定**
5. **実行前に「〜しますっぴ」と簡潔に伝える**（危険な操作以外）

---

## 🔎 Web + X 統合リサーチ（2026-02-16更新）

### トリガーフレーズ
- "〜についてリサーチして"
- "〜を調べて"
- "〜の情報を教えて"
- "〜について調査して"
- "〜の最新情報は？"
- "〜を詳しく"

### 動作（Web + X両方で検索）
1. **Web検索**: web_search ツールでWeb検索（Brave Search API）
   - 上位10件の結果を取得
   - タイトル、URL、スニペットを分析
2. **X検索**: bird CLI でX（Twitter）検索
   - 最新10件のツイートを取得
   - ユーザーの評判・感想を収集
3. **統合報告**: 両方の結果を統合して要約
   - Web: 公式情報・使い方・技術情報
   - X: ユーザーの生の声・評判・体験談
   - 参考URLを提示

### 使用例
**ユーザー:** "Soraのウォーターマーク除去方法についてリサーチして"
**リッキー:** 
1. web_search("Sora watermark removal") → Web検索
2. bird search "Sora watermark removal" → X検索
3. 統合報告:
   ```
   Soraのウォーターマーク除去について調べましたっぴ！🔍
   
   【Web検索結果】
   - 方法1: ...（公式手順）
   - 方法2: ...（サードパーティツール）
   
   【Xでの評判】
   - @user1: "実際に試してみた結果..."
   - @user2: "このツール便利だった"
   
   参考URL: ...
   ```

**ユーザー:** "最新のAI動画編集ツールについて調べて"
**リッキー:**
1. web_search("AI video editing tools 2025") → 公式情報
2. bird search "AI video editing tools" → ユーザー評価
3. おすすめツールを紹介（Web + X統合）

### 自動判断ルール
- **「リサーチ」「調べて」「調査」** → 即座に Web + X両方で検索
- 検索キーワードを自動抽出
- 両方の結果を読みやすく統合（生データを垂れ流さない）
- 公式情報（Web）+ ユーザーの声（X）を組み合わせる
- 関連URLを提示

- **X用**: 720p, 2500k

### 処理時間
- 画質改善: 数分（動画の長さによる）
- 音声改善: 数分（Adobe Podcast処理含む）

## 🎯 自動判断ルール

### 優先順位
1. **明示的なキーワード** → 対応するSkill使用
2. **文脈から推測** → 適切なSkillを選択
3. **不明な場合** → 確認してから実行

### 例外処理
- パスワード/APIキーの扱い → 必ず確認
- 外部投稿 → 確認してから実行
- ファイル削除 → 必ず確認

### 報告形式
- 実行前: 「〜を実行しますっぴ」（確認不要なタスク）
- 実行後: 結果を自然に会話に組み込む
- エラー時: 分かりやすく説明

## 📌 使用頻度の学習

**よく使うパターン:**
- メモ作成 → obsidian-vps
- X検索 → bird
- URL要約 → summarize

**あまり使わない:**
- GIF検索 → 明示的リクエスト時のみ
- パスワード取得 → セキュリティ上、明示的リクエストのみ

## 🔧 新しいSkill追加のプロセス

### いつ追加するか
- タスクを実行する際、既存のSkillsで不十分な場合
- 便利そうなツール・CLIを見つけた場合
- andoさんの要望から新機能が必要な場合

### 追加手順
1. **インストール**: 依存関係（CLI/パッケージ）をインストール
2. **テスト**: 動作確認
3. **Skill作成**: `/root/clawd/skills/<name>/SKILL.md` を作成
4. **トリガー登録**: このファイル（SKILL_TRIGGERS.md）に追加
5. **動作確認**: 会話トリガーでテスト

### 追加後のルール
- **即座に使用開始**: 追加したSkillは即座にトリガー可能
- **自然な統合**: 会話の流れで自然に使う
- **ドキュメント更新**: TOOLS.mdに使用方法を記録
