# トレーディングダッシュボード チャート設計提案

**作成日**: 2026-02-15  
**対象**: Bitget自動トレーディング

---

## 📊 推奨チャート構成

### 1️⃣ 資金推移グラフ（折れ線）【最重要】

**表示内容**:
- X軸: トレード番号（0, 1, 2, 3...）
- Y軸: 資金残高（$10,000 → 現在）

**効果**:
- ✅ 資金の増減が一目でわかる
- ✅ パフォーマンスの良し悪しが明確
- ✅ 初期資金からの推移を可視化

**データ例**:
```
トレード0: $10,000（初期）
トレード1: $8,000（LQTY エントリー）
トレード2: $6,400（MANTA エントリー）
トレード3: $6,336（TNSR 決済 -$64）
トレード4: $5,068.80（STG エントリー）
```

---

### 2️⃣ 累積損益グラフ（折れ線）

**表示内容**:
- X軸: トレード番号
- Y軸: 累積損益（$）
- ゼロライン: 損益0のライン

**効果**:
- ✅ トータルの損益推移が明確
- ✅ ゼロラインで損益判断が容易
- ✅ 利益/損失のトレンドがわかる

**データ例**:
```
トレード0: $0
トレード1: $0（エントリーのみ）
トレード2: $0（エントリーのみ）
トレード3: -$64（損失確定）
トレード4: -$64（エントリーのみ）
```

---

### 3️⃣ 日別PnLグラフ（棒グラフ）

**表示内容**:
- X軸: 日付（2026-02-14, 15, 16...）
- Y軸: 日別損益（$）
- 色: プラス=緑、マイナス=赤

**効果**:
- ✅ 日ごとのパフォーマンスが一目瞭然
- ✅ 良い日/悪い日が明確
- ✅ トレンドの変化がわかる

**データ例**:
```
2026-02-14: -$64（赤）
2026-02-15: $0（まだトレード完了なし）
```

---

### 4️⃣ 銘柄別PnLグラフ（横棒グラフ）

**表示内容**:
- X軸: 損益（$）
- Y軸: 銘柄名
- 色: プラス=緑、マイナス=赤

**効果**:
- ✅ どの銘柄が良い/悪いか比較
- ✅ 得意な銘柄がわかる
- ✅ 避けるべき銘柄がわかる

**データ例**:
```
TNSRUSDT: -$64（赤）
LQTYUSDT: $0（オープン中）
MANTAUSDT: $0（オープン中）
STGUSDT: $0（オープン中）
```

---

### 5️⃣ 勝率円グラフ

**表示内容**:
- Win/Loss/Open の割合
- 色: Win=緑、Loss=赤、Open=黄

**効果**:
- ✅ トレード成功率が視覚的
- ✅ 勝率がすぐわかる
- ✅ オープンポジションの割合もわかる

**データ例**:
```
Win: 0件（0%）
Loss: 1件（25%）
Open: 3件（75%）
```

---

### 6️⃣ 資金配分円グラフ

**表示内容**:
- 未使用資金 vs ポジション使用中
- 色: 未使用=青、使用中=黄

**効果**:
- ✅ リスク管理状況がわかる
- ✅ ポジションサイズが適切かわかる
- ✅ 資金の余裕度がわかる

**データ例**:
```
未使用資金: $5,068.80（51%）
ポジション使用中: $4,931.20（49%）
```

---

## 💡 最優先で作成すべきチャート（TOP 3）

### 1. 資金推移グラフ（折れ線）
**理由**: トレーディングで最も重要な指標。資金が増えているか減っているかが一目でわかる。

### 2. 累積損益グラフ（折れ線）
**理由**: パフォーマンスの確認に必須。ゼロラインで損益判断が容易。

### 3. 日別PnLグラフ（棒グラフ）
**理由**: 日々の成果確認に最適。良い日/悪い日が明確。

---

## 🎯 実装方法

### 手動作成（推奨）
1. Googleスプレッドシートのグラフ機能を使用
2. Chartsシートのデータを元にグラフを作成
3. Dashboardシートに配置

### 自動作成（高度）
- Google Sheets API を使用してグラフを自動生成
- データ更新時に自動的にグラフも更新
